# 🧾 US12 - Détection de faux billets

## Contexte
Ce projet porte sur la détection de faux billets à partir de mesures physiques. Il met en oeuvre une démarche complète de machine learning supervisé pour distinguer les billets authentiques des billets frauduleux.

Il s'inscrit naturellement dans une logique de prévention du risque, avec un lien fort avec la fraude, la sécurité et l'analyse de signaux suspects.

## Problématique métier
Comment construire un modèle capable d'identifier automatiquement les faux billets tout en expliquant les critères qui influencent la décision ?

## Objectifs
- Explorer les variables discriminantes entre vrais et faux billets.
- Préparer les données pour une tâche de classification.
- Tester et comparer des approches de modélisation.
- Évaluer la performance avec des métriques adaptées au risque métier.
- Expliquer les résultats de manière claire pour faciliter la confiance dans le modèle.

## Démarche analytique
- Analyse exploratoire des dimensions des billets.
- Recherche de corrélations et de variables discriminantes.
- Préparation des jeux d'entraînement et de test.
- Entraînement de modèles de classification.
- Évaluation via matrice de confusion et indicateurs de performance.
- Interprétation des erreurs, notamment les faux positifs et faux négatifs.

## Compétences démontrées
- Machine learning supervisé.
- Classification binaire.
- Préparation de données avec Python.
- Évaluation de modèles : accuracy, précision, rappel, matrice de confusion.
- Analyse du risque d'erreur selon le contexte métier.
- Communication claire des résultats d'un modèle prédictif.
