# 🎯 Proof Point : P10 — Eau potable

## 1️⃣ Contexte
Le projet vise à produire un support décisionnel sur l’accès à l’eau potable, la stabilité politique et la mortalité WASH.

## 2️⃣ Données
- **Sources** : cinq sources de données citées dans le README.
- **Qualité** : standardisation des pays et construction de tables étoile.
- **Limites** : la mortalité WASH est limitée temporellement et certaines analyses récentes doivent être interprétées avec prudence.

## 3️⃣ Démarche
- Préparation de données en Python et Langage M.
- Modèle canonique et tables fact / dim pour Power BI.
- Documentation métier et template de restitution.

## 4️⃣ Résultats + Impact
- Modèle Power BI `.pbit`.
- Tables étoile exportées pour l’analyse.
- Trois vues métier et trois scores composites structurants.

## 5️⃣ Limites & Pistes
- Poursuivre les analyses temporelles avec prudence.
- Compléter la documentation des KPI finaux.
- Éventuellement enrichir avec de nouvelles sources.
