# 🛒 US2 - Analyse e-commerce

## Contexte
Une entreprise e-commerce souhaite analyser ses ventes pour identifier les tendances et optimiser ses performances.

## Objectifs
- Explorer les données de ventes
- Identifier les produits performants et sous-performants
- Détecter les tendances saisonnières
- Proposer des recommandations business

## Données utilisées
- Historique des ventes
- Produits
- Clients
- Dates et périodes

## Livrables
- Notebook d analyse
- Visualisations
- Recommandations métier
