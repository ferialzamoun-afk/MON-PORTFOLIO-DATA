# EPIC 3 — SQL & Bases de données

En tant qu'analyste, je veux requêter une base SQL afin d'extraire et analyser les données.

---

## 🎯 Objectifs de l’EPIC
- Concevoir des bases de données normalisées
- Manipuler SQL pour extraire et transformer les données
- Comprendre les enjeux RGPD liés aux données
- Produire des analyses basées sur des requêtes robustes

---

## 📌 User Stories incluses

### **US3 — Requêtes SQL**
- **En tant que** analyste  
- **Je veux** requêter une base SQL  
- **Afin de** extraire et analyser les données  
- **Compétences :** SQL, gestion BD  
- **Durée :** 1,25 semaines

### **US5 — Base immobilière SQL**
- **En tant que** analyste immobilier  
- **Je veux** créer et utiliser une base immobilière  
- **Afin de** explorer les données et produire des analyses fiables  
- **Compétences :** 3NF, création tables, import CSV, RGPD, SQL avancé  
- **Durée :** 2,5 semaines
