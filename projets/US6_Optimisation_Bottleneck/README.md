# 🛒 US6 - Optimisation des données d une boutique

## Problème
Une boutique en ligne utilise plusieurs systèmes (ERP, web, gestion), avec des écarts de prix, de stocks et de référentiels.

## Solution
Ce projet unifie et nettoie les données pour fournir une vue fiable et faciliter les décisions métier.

## Résultats attendus
- Données consolidées et cohérentes
- Détection des anomalies
- Recommandations d actions
